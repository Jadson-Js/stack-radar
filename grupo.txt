Fala pessoal, Queria compartilhar um projeto open-source que fiz para tirar uma dúvida interessante: o que o mercado realmente está pedindo.

Um amigo me aconselhou a anotar as stacks de cada vaga que eu aplicava para ver as tendências. Achei isso chato de se fazer na mão então resolvi para uma solução em escala industrial.

Criei um script em Python que consome dados de uma api que serve de banco de vagas. o código faz buscas por termos (desenvolvedor, web developer, etc.) e cruza as palavras-chave direto nas descrições das vagas para extrair as reais stacks exigidas.

O que eu consegui mapear:

72.965 vagas analisadas no total (de 23/06/2025 até 23/06/2026).

16.944 vagas no Brasil e 56.021 nos EUA.

Infelizmente, para o nosso desespero, 87% das vagas são presenciais 🤡.

Liderando as menções: Machine Learning, Oracle, Java, AWS e C.

Os dados são ouro para quem tá meio perdido nos estudos. Eu mesmo decidi focar mais em Java para abrir meu leque de oportunidades depois de ver esses números.

O projeto é aberto e queria convidar vocês para dar uma olhada, usar os insights ou contribuir! Toda ajuda é bem-vinda, seja revisando código, melhorando a busca, criando novos gráficos ou corrigindo bugs.
